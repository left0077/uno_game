# 🔍 Uno 规则审查报告

> 审查时间：2026-03-15  
> 审查范围：GAME_RULES.md + UI_DESIGN.md  
> 发现：6处不合理/待澄清的问题

## 🚨 不合理问题

### 1. 多牌同出+反转的逻辑矛盾

**问题描述：**
规则说"反转效果在打出该牌时立即生效，后续牌的出牌顺序可能改变"

**矛盾点：**
- 多牌同出是一次性打出多张相同数字的牌
- 如果其中包含反转牌，反转效果"立即生效"意味着顺序改变
- 但后续牌（同一次多牌同出）的"出牌顺序"已经确定了

**建议修复：**
```
修订规则：
多牌同出包含反转牌时，反转效果在**所有牌打出后**生效，
影响的是**下一轮**的出牌顺序，而非当前多牌同出过程。
```

---

### 2. 抢打与多牌同出的时间冲突

**问题描述：**
规则说"玩家B多牌同出时，出第一张后，其他玩家可以抢打第二张"

**不合理点：**
- 多牌同出是**一次性操作**（玩家选择多张牌后点击"出牌"）
- "出第一张后"意味着有一个时间窗口，但多牌同出没有中间状态
- 如果理解为动画分步展示，2秒抢打窗口会在动画中随机出现，玩家无法预判

**建议修复：**
```
选项A：禁止多牌同出过程中的抢打
- 多牌同出视为一次性操作，期间不可被抢打
- 简化实现，避免时间冲突

选项B：抢打只针对第一张
- 多牌同出的第一张牌可以被抢打
- 如果被抢打，多牌同出被打断，原玩家只出剩余的牌
```

---

### 3. AI困难难度100%抢打过于强势

**问题描述：**
困难AI "100%抢打（有牌时）"

**不合理点：**
- 100%抢打会让真人玩家几乎没有出牌机会
- 游戏变成"谁手牌数字好谁赢"，失去策略性
- 体验极差：玩家刚出一张，AI立刻抢打，玩家连续被跳过

**建议修复：**
```
修订策略：
| 难度 | 抢打策略 |
|------|---------|
| 🟢 简单 | 从不抢打 |
| 🟡 普通 | 30%概率抢打（有牌时） |
| 🔴 困难 | 70%概率抢打（有牌时），但每轮最多抢打1次 |

增加限制：
- 任何玩家每轮最多被抢打1次（防止连续被抢）
- 或者：抢打后有3秒冷却期，该玩家不可被再次抢打
```

---

### 4. 智能排序与"无可出牌"状态的矛盾

**问题描述：**
UI设计中的智能排序"可出牌在前+推荐度排序"，同时不可出牌"置灰50%+透明度70%"

**矛盾点：**
- 如果玩家**没有可出牌**（所有牌都不可出），智能排序后所有牌都是灰色的
- 用户可能困惑：为什么排在前面还是灰色的？
- 与"一目了然看出哪些牌能打"的目标矛盾

**建议修复：**
```
修订UI规则：
1. 当没有可出牌时，显示提示"无牌可出，请摸牌"
2. 所有牌保持正常颜色（不禁用），但点击后提示"无可出牌，请摸牌"
3. 或者：智能排序在无可出牌时，显示"🎯 无可出牌，请点击摸牌"
```

## ⚠️ 待澄清问题

### 5. 抢打与叠加规则的衔接模糊

**问题描述：**
规则说"抢打一张+2/+4可触发叠加，抢打者可以继续出+2/+4来叠加"

**模糊点：**
- "继续出"是指从手牌再出一张，还是指"抢打"本身可以叠加？
- 如果是从手牌出，那是抢打者的正常回合吗？
- 抢打后抢打者获得回合权，此时叠加是否重置？

**建议澄清：**
```
修订规则：
抢打一张+2/+4后：
1. 抢打者可以选择"继续叠加"：从手牌再出一张+2/+4（如果是同类型）
2. 或选择"停止叠加"：轮到原下家（按抢打后的新顺序）选择接牌或摸牌
3. 叠加计数器累加，直到某玩家无法接牌时摸取累计张数
```

---

### 6. 抢打与质疑窗口的时间计算

**问题描述：**
规则说"抢打不影响质疑窗口，质疑窗口仍按原出牌顺序计算"

**疑问：**
- 玩家A出牌 → 玩家B抢打 → 玩家A的质疑窗口是否还在？
- 如果B抢打成功，A的回合被跳过，A还能质疑B吗？
- "按原出牌顺序计算"是指按抢打前的顺序，还是抢打后的新顺序？

**建议澄清：**
```
修订规则：
抢打与质疑窗口的处理：
1. 玩家A出倒数第2张后，质疑窗口开启
2. 玩家B抢打成功，质疑窗口保持开启（针对A的UNO）
3. 抢打者B的质疑窗口从B出倒数第2张时开启
4. 如果A在抢打后立即出完牌（只剩1张被抢打），质疑窗口立即关闭
```

## 📝 其他建议

### 7. 手牌上限与多牌同出的配合

**现状：**
手牌理论无上限，但多牌同出要求"相同数字"

**建议：**
- 当手牌超过20张时，智能排序优先显示"可多牌同出"的组合
- 帮助玩家快速减少手牌数量

### 8. 抢打按钮的触发时机

**现状：**
"2秒内点击抢打按钮"

**建议：**
- 增加视觉提示：其他玩家出牌时，如果自己有可抢打的牌，手牌区显示"⚡可抢打"提示
- 这样玩家不用盯着屏幕等2秒窗口
