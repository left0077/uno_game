### 4. Out 模式

| # | 功能 | 测试点 |
|---|------|--------|
| O1 | 开局 | 每人7张、PhaseTimer 显示 Phase 0、手牌上限20 |
| O2 | 连打-对子 | 2张同色/同值可选、打出清2张、无惩罚 |
| O3 | 连打-三条 | 3张同色可选、打出清3张、下家跳过 |
| O4 | 连打-彩虹 | 4色同数字可选、指定目标+3 |
| O5 | 连打-顺子 | 3+张同色连续可选、下家摸N-2张 |
| O6 | 万能牌不能参与连打 | wild/+4 选中时不触发连打 |
| O7 | 手牌上限淘汰 | 摸牌后>20淘汰、排名填充 |
| O8 | 阶段推进 | 3分钟→Phase1注入+3、6分钟→Phase2注入+5、9分钟→Phase3注入+8 |
| O9 | 阶段计时器 | 显示当前Phase、距下一阶段倒计时、颜色预警 |
| O10 | 惩罚卡叠加 | +2→+3→+4→+5→+8 跨类型叠加链 |
| O11 | 反转弹回 | 被惩罚时出反转→弹回来源、来源可继续反制 |
| O12 | 20分钟超时 | 全局超时强制结束 |
