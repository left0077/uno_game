## 🎯 v2.0 核心特性

1. **Action API v2.0** - 服务器权威的游戏动作接口
2. **Out 模式** - 大逃杀玩法，支持连打和彩虹转移
3. **连打系统** - 对子/三条/彩虹/顺子组合出牌
4. **反转反击** - 用反转牌弹回惩罚
