# 🐛 Bug 修复记录

此目录用于记录每个版本的 Bug 修复情况。

## 目录结构

```
docs/bug-fixes/
├── README.md              # 本文件
├── TEMPLATE.md            # 修复记录模板
├── v1.0.0-plan.md         # v1.0.0 版本修复计划（当前）
├── v1.0.0.md              # v1.0.0 实际修复记录
├── v1.0.1-plan.md         # v1.0.1 版本修复计划
├── v1.0.1.md              # v1.0.1 实际修复记录
└── ...
```

## 版本命名规则

- **主版本号**：重大功能更新或架构重构
- **次版本号**：新功能或重要规则变更
- **修订号**：Bug 修复和小优化

## 文档类型

| 文件名后缀 | 说明 |
|-----------|------|
| `-plan.md` | 修复计划，包含待修复的 Bug 列表 |
| `.md` | 实际修复记录，包含已修复的内容和验证结果 |
