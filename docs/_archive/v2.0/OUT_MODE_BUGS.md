## 详细描述

### 问题1: 牌堆图标错位
**现象**: "146张 点击摸牌"文字和牌堆图标位置重叠/错位
**截图**: ![image1]

### 问题2: UI元素太多
**现象**: 界面拥挤，信息过载
**建议**: 精简显示，合并冗余信息

### 问题3: 可出牌提示太多
**现象**: 每张可出牌下面都显示提示（如"颜色匹配: blue"、"万能牌可以出"）
**建议**: 只保留必要的提示，或hover时显示

### 问题4: 对子出牌显示错误 ❗
**现象**: 选中的对子牌中，不同颜色的牌显示为灰色（不可出状态），但实际应该可以出
**截图**: ![image2] - 黄色4显示灰色但应该可选
**影响**: 用户误以为不能出对子

### 问题5: 胜利玩家继续抽牌
**现象**: 已经出完牌的玩家还会被惩罚抽牌
**预期**: 胜利玩家应该不再参与游戏

### 问题6: 缺少缩圈倒计时
**现象**: Out模式缩圈前没有倒计时提示
**预期**: 应该在缩圈前显示倒计时

### 问题7: 缩圈提示不符规则书
**现象**: 缩圈提示文字/样式不符合规则书规范
**预期**: 按照规则书显示缩圈信息
