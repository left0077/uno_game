## 严重问题 (Critical)

### ❌ 问题1: 类型定义重复
**位置**: `shared/actionApi.ts` vs `server/src/shared/actionApi.ts`
**问题**: 两个文件内容相同，需要同步维护
**风险**: 修改一处忘记修改另一处导致类型不一致
**解决**: 使用符号链接或构建时复制

### ❌ 问题2: 前端硬编码规则未完全移除
**位置**: `client/src/pages/Game.tsx:145-165`
**问题**: 仍有硬编码的出牌规则计算
**风险**: 前后端规则可能不一致
**解决**: 完全依赖服务端返回的availableActions

### ❌ 问题3: Socket事件名不一致
**位置**: `server/src/socket/SocketHandler.ts` vs `client/src/hooks/useSocket.ts`
**问题**: 服务端发送 `message:received`，客户端监听 `RECEIVE_MESSAGE`
**风险**: 表情消息无法正常显示
**解决**: 统一事件名

## 高优先级问题 (High)

### ⚠️ 问题4: AI策略缓存未清理
