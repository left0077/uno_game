## 二、自动化测试

### 2.1 快速运行所有测试

```bash
# 运行所有测试（单元测试 + E2E测试）
npm test              # 单元测试
npm run test:e2e      # E2E测试
npm run test:all      # 全部测试
```

### 2.2 单元测试

```bash
# 后端单元测试
cd server
npm run build
node test-card.mjs      # CardManager 测试
node test-room.mjs      # RoomManager 测试
node test-game.mjs      # UnoGame 测试

# 前端构建测试
cd client
npm run build
```

### 2.3 E2E 测试

```bash
cd e2e
npm install
npx playwright install chromium

# 运行所有 E2E 测试
npm test

# 运行特定测试
npx playwright test basic.spec.ts      # 基础功能
npx playwright test reconnect.spec.ts  # 断线重连
npx playwright test gameplay.spec.ts   # 游戏流程
npx playwright test features.spec.ts   # 特色功能

# 调试模式
npm run test:ui       # UI模式
npm run test:headed   # 可见浏览器
```
