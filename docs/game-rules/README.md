## 快速开始

1. 创建房间
2. 选择游戏模式（标准/Out）
3. 邀请好友或添加AI
4. 开始游戏！

## 通用规则

无论哪种模式，以下规则始终适用：

- 回合限时120秒
- 超时自动摸牌
- 支持2-8人游戏
- AI难度可选（简单/普通/困难）
